When Agents make changes they should follow
- the code style guidelines in [docs/code-style.md](docs/code-style.md)
- the renderer addition process in [docs/adding_a_new_renderer.md](docs/adding_a_new_renderer.md)

Every PR (including documentation, workflow, and dependency changes) **must** add an entry to [docs/CHANGELOG.md](docs/CHANGELOG.md) under `Upcoming Changes`. See the [Changelog section of the code style guide](docs/code-style.md#changelog) for details.
